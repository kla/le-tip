Le tip

An extension of the bootstrap-tooltip plugin with the following new features added:

  • auto placement of tooltip within viewport
  • additional placement positions: top-left, top-right, bottom-right, bottom-left
  • tooltips can be dismissed by pressing escape or clicking outside of the tooltip
  • can add a css class to the tooltip

Basic Usage

  • $(".tip").tooltip();
  • <span class="tip" title="basic tooltip with placement determined automatically">
  • <span class="tip" title="top-left placement" data-placement="top-left">

Adding a CSS class

  • $(".tip").tooltip({cssClass: "orange"});
