Le tip

An extension of the bootstrap-tooltip plugin with the following new features added:

  • auto placement of tooltip within viewport
  • additional placement positions: top-left, top-right, bottom-right, bottom-left
  • tooltips can be dismissed by pressing escape or clicking outside of the tooltip
  • add a css class to the tooltip
  • hover intent so tooltips aren't unnecessarily triggered
  • ajax tooltips

Basic Usage

  • $(".tip").tooltip();
  • <span class="tip" title="basic tooltip with automatic placement">
  • <span class="tip" title="top-left" data-placement="top-left">
  • <span class="tip" title="adding a css class" data-css="orange">
  • <span class="tip" data-source="ajax-tip.html" data-html="true">
